
# Descripción

    Programa Hello World escrito en C23.

    Compilador              GCC
    Versión del compilador  15.2.0
    Versión de C            C23    
        
# Prueba del compilador

    Cree el archivo "hello.c" que envía por stdout el mensaje "Hello World"

# Compilación desde la línea de comandos

    Para compilar use el comando "gcc -std=c23 hello.c -o hello"

    GCC --> indica el compilador  
    -std=c23 --> usa el estandar c23
    hello.c --> es el archivo fuente
    -o hello --> el ejecutable se llama hello

# Creacion de txt

    ./hello > output.txt

# Credito Extra con Make

    Para compilar: mingw32-make
    Para ejecutar: mingw32-make run
